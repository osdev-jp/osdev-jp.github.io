# チャットのチャンネル案内

| チャンネル名    | 役割 |
|-----------------|------|
| #自己紹介       | 自己紹介用 |
| #eventわいわい  | 自作OSもくもく会などの各種イベントの告知、イベント中の会話 |

| #雑談なんでも   | 雑談用（技術に関連しない話題を何でも扱うところ） |
| #初心者質問相談 | 初心者専用の質問・相談チャンネル |
| #進捗発表       | 自作OS開発や低レイヤ関連の進捗発表 |
