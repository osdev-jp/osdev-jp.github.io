## Discord、メーリングリストへの参加方法
Discordやメーリングリスト（ML）ではOS開発の質問とか相談が日々やり取りされています。
「30日でできる！OS自作入門」や「ゼロからのOS自作入門」のサポートも有志により行われています。

DiscordかMLのいずれか（または両方）へ参加いただくことでosdev-jpのメンバーとなれます。
詳しくは [参加方法](joinus.md) をご覧ください。
